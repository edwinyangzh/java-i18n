The ServletResponse.getOutputStream() method returns a ServletOutputStream suitable for writing binary data in the response. However, in most usesthe response data is text and not binary data. For text, developers should use ServletResponse.getWriter().

Noncompliant Code Example

class ServletResponseGetOutputStreamCheck {

  ServletOutputStream methodA1(ServletResponse servletResponse) {
    return servletResponse.getOutputStream();  //Noncompliant
  }
}

Compliant Solution

ServletResponse.getWriter()
