String.compareTo() and String.compareToIgnoreCase() compare two strings based on the Unicode value of each character in the strings and not the different ordering they is required for different locales. Results will not linguistically be correct, especially for non-English string comparisons.

Noncompliant Code Example

  String1.compareTo(String2)
  ...or...
  String1.compareIgnoreCase(String2)

Compliant Solution

  Locale userPreferredLocale = Locale.forLanguageTag("zh-Hans");
  ...
  Collator collatorForCompare = Collator.getInstance(userPreferredLocale);
  collatorForCompare.compare(String1, String2)
